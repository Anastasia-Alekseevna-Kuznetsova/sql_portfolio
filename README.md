# 💰 SQL-портфолио

Добро пожаловать в мою коллекцию SQL-запросов для реальных бизнес-задач. Здесь собраны примеры кода, которые я использовала в аналитической работе (банк, финтех, консалтинг).

## 🎯 Цель репозитория
Показать владение SQL на уровне, достаточном для:
- агрегации и оконных функций
- когортного анализа и LTV
- построения витрин данных для дашбордов

## 📂 Структура

| Файл | Описание | Что используется|
|------|----------|----------------|
| `financial_metrics.sql` | Маржинальность продуктов, топ-продукты по росту | оконные функции, LAG, джойны, CTE |
| `cohort_retention.sql` | Удержание клиентов по когортам | DATE_TRUNC, оконные функции, удержание |
| `customer_lifetime_value.sql` | LTV по сегментам клиентов | CASE, подзапросы, группировки |
| `product_margin_analysis.sql` | Динамика маржинальности по месяцам | LAG, PARTITION BY, процентное изменение |

## 🛠 Используемый диалект
PostgreSQL (совместимо с Redshift, Greenplum). При необходимости адаптирую под другой SQL.
